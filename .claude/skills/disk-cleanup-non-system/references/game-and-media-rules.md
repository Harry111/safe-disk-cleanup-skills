# 游戏和素材保护规则

发现以下类型目录时默认保护：

- Games、Steam、Epic、Tencent、miHoYo 等游戏目录；
- 素材、视频、照片、图片、录屏、剪辑、工程文件；
- 模型、音频、训练数据、工程资源；
- 剪辑软件素材库（如 JianyingPro Materials、CapCut Materials、Premiere Media Cache）；
- 剪辑软件草稿项目文件（注意：.recycle_bin 和 .cloud_cache 是缓存/回收站，可清理，但草稿本体保护）；
- Backup、Archive、备份、资料、论文、报告。

如果目录明显是游戏或素材，不需要继续深入分析每个文件。直接输出目录总大小、类型判断和“需用户确认”。
