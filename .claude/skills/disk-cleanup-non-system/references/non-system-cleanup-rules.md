# 非系统盘清理规则

非系统盘清理的重点是：指定盘符内分析、分类、确认后处理。不要跨盘符，不要自动删除用户资料。

## 默认保护

以下内容默认保护，不自动删除：

- 游戏目录；
- 项目源码；
- Git 仓库；
- 照片、视频、素材；
- 论文、报告、文档；
- 数据库文件；
- 虚拟机文件；
- 备份目录；
- 聊天记录；
- 用户手动命名的资料目录；
- 未知目录。

## 可建议清理

可以建议清理但需要按风险确认：

- 临时目录；
- 旧安装包；
- 旧日志；
- 浏览器缓存；
- 应用纯缓存；
- node_modules；
- venv；
- dist/build/target；
- 包管理器缓存。
